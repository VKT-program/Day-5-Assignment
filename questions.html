Q1. How to set justify property in the paragraph element?

Answer:    This is the way how the text text looks in justify using the syntax: "text-align:justify;"
In CSS, `justify` is a value for the `text-align` property. When `text-align: justify;` is applied to a block of text (like a paragraph), it makes the text align along both the left and right margins. This is achieved by the browser automatically adjusting the spacing between words and sometimes letters, to ensure that each line fills the available width evenly, creating a straight, flush edge on both sides of the text block. This formatting is commonly seen in newspapers, magazines, and books, giving content a more formal and structured appearance.

Q2. How to add shadow to text?

Answer:    This is the way how the text look with a shadow. This is done using the syntax:"text-shadow:(any value accordingly)"
The `text-shadow` CSS property adds a shadow effect to text. It allows you to specify the shadow's horizontal offset, vertical offset, blur radius, and color. This property can be used to create various visual effects, such as making text appear raised, embossed, or glowing, or to simply add subtle depth for improved readability against complex backgrounds.

Q3. How to specify all the font properties in one declaration?

You can specify many common font properties in one declaration using the CSS font shorthand property. This is a powerful shorthand that allows you to set font-style, font-variant, font-weight, font-stretch, font-size, line-height, and font-family all in a single line. The order of values in the font shorthand is crucial:
font: font-style font-variant font-weight font-stretch font-size/line-height font-family;
